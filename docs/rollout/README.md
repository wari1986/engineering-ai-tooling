# Rollout plan

## Phases

1. **Pilot** – one product repo with known team.
2. **Expand** – additional repos using reusable workflows and Codex pack.
3. **Standardize** – enforce governance requirements.
4. **Operate** – quarterly policy review, update training.

## Success criteria

- Governance owners approve baseline.
- Product repos adopt Codex config pack.
- Metadata-only telemetry policy validated.
- Workflow templates integrated.
