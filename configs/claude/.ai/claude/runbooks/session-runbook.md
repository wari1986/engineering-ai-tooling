# Session Runbook

## Pre-flight checklist

- Confirm project settings and available tools.
- Review `CLAUDE.md` and imported context.
- Open active task brief for multi-session work.

## Troubleshooting workflow

1. Reproduce.
2. Verify environment assumptions.
3. Consult architecture/decision docs.
4. Identify root cause and patch minimally.

## Validation and readiness

- Run targeted lint/test/build checks.
- Validate docs and templates stayed aligned.
- Record validation notes for review.

## Safety notes

- Avoid destructive commands unless explicitly required.
- Preserve placeholders and never introduce secrets.
