# Settings templates

Claude Code uses scoped settings files. Copy these templates into the locations below:

- User (global): `~/.claude/settings.json`
- Project (shared): `.claude/settings.json`
- Local (unshared): `.claude/settings.local.json`
- Managed (IT-managed): system-level `managed-settings.json`

Templates in this folder:

- `global-settings.example.json` – baseline user settings example.
- `project-settings.example.json` – shared repo settings and hooks example.
- `mcp.project.example.json` – project-scoped MCP server configuration example.
